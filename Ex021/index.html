exemplo de caixa box level

paragráfos sao exemplos de box level, mas os links são exemplos de in-line level

Um site é feito de caixas, por isso aprenda bem o conceito de in-line e de box level

O paragrafo por exemplo é box level porém os links são inline

com o comando display você consegue mudar de box pra inline e vice versa

Usar a ferramenta dev tools do google para guiar em relação a margin, padding, outline,border.
